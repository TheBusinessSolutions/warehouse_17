Lots Expiration Dashbard
=====================
Lots Expiration Dashbard and hide search and filter at this

Configuration
=============
* To Hide Search View
    1) Open Developer Mode
    2) Go To Views
    3) Search view 'expired_lots.res_config_settings_view_form'
    4) Change id=hide_filters_groupby to invisible =0
    5) Go To General Setting
    6) Choose Custom Model => expired.lots model at Hide Filters and Group By
    6) Refresh and change to invisible =0  at view 'noi_expired_lots.res_config_settings_view_form'
