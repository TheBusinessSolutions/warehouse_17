Manually configuring the carrier for each picking is inefficient,
especially when it's already predefined for the partner or their primary
partner. This add-on automates this process for companies and allows you
to choose whether or not to automate it, based on the type of operation.

Configuration
=============

Configure auto assign carrier for company:
------------------------------------------

1. Go to Inventory > Configuration > Settings > Section Shipping

2. Find "Carrier auto assign" and enable it

   |ENABLE_AUTO_ASSIGN_CARRIER_FOR_COMPANY|

Please note that enabling this setting will activate the "Carrier auto
assign" option for all types of operations. If you do not wish to use it
for a particular type of operation, disable it.

Configure auto assign carrier for picking type:
-----------------------------------------------

1. Go to Inventory > Configuration > Operation types

2. Find "Carrier auto assign" and enable it

   |ENABLE_AUTO_ASSIGN_CARRIER|

Usage
=====

Configure shipping method in the contact
----------------------------------------

1. Go to Contacts or any other menu to access contacts.
2. Select the contact you want to configure the shipping method.
3. Go to the tab "Sales & Purchases" and select the shipping method.

|SHIPPING_METHOD_CONTACT|

Set shipping method on the picking
----------------------------------

1. Go to Inventory > Operations > Receipts

2. Create a new picking

3. Select the Receive From and the Shipping Method field will
   automatically be filled in on the Additional Information tab.

   |SHIPPING_METHOD_PICKING|

Set shipping method when creating a sales order
-----------------------------------------------

1. Go to Sales > Orders > Quotations

2. Create a new sale order

3. Add the partner, lines, and other necessary data, then confirm the
   sales order.

4. A picking will be created with the Carrier configured in the partner
   selected in the sales order.

   |SALE_SHIPPING_METHOD_PICKING|

Please note that for sections `Set shipping method on the
picking <#set-shipping-method-on-the-picking>`__ and `Set shipping
method when creating a sales
order <#set-shipping-method-when-creating-a-sales-order>`__, the order
of priority is as follows:

1. Delivery method of the primary contact (parent_id).
2. Delivery method of the contact.
