Stock Card Report

An essential tool for tracking the complete history of your inventory. Generate detailed Stock Card / Product Card reports with opening balances, in/out movements, and closing balances for any date range.

Key Features

  1. Flexible Date Range

    Generate reports for any custom date period with accurate opening and closing balances.

  2. Location Filtering

    Filter by specific warehouse locations with optional child location support.

  3. Product & Category Filtering

    Select specific products or filter by product categories to focus your analysis.

  4. Running Balances

    Track quantity and value balances with each transaction for complete visibility.

  5. Valuation Support

    Support for Standard and Average costing methods with value tracking.

  6. Export Options

    Generate professional PDF reports or export to Excel for further analysis.
